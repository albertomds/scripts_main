# pDynamo3 Tutorials

These tutorials provide lightweight templates for organizing pDynamo3 workflows
inside `CODEX_allScripts`.

The scripts are placeholders. They may require edits depending on the local
pDynamo3 installation, Python environment, input files, and available model
libraries.

## Citation

When using pDynamo3 in real calculations, cite the relevant pDynamo3
publications and any methods, force fields, quantum chemistry models, molecular
mechanics parameters, or supporting software used in the workflow.
