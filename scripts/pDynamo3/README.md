# pDynamo3 Script Templates

This folder contains lightweight pDynamo3 tutorial templates.

The scripts are intentionally small and do not ship with molecular structures,
trajectories, binary files, or unpublished research data. Copy a tutorial folder,
add your own input files under `inputs/`, and edit the variables marked near the
top of each `run_example.py`.

## Tutorials

- `01_first_system/`: template for checking a first pDynamo3 system workflow.
- `02_qc_energy_placeholder/`: placeholder for a QC single-point energy setup.

## Important warning

pDynamo3 installations differ across machines. Module names, import paths,
environment variables, and available data files may need local edits before a
template can run as a real calculation.

## Citation

If you use pDynamo3 in a project, cite the relevant pDynamo3 publications and
any methods, force fields, quantum chemistry models, or molecular mechanics
parameters used in your calculation.
