# pDynamo3 Examples

These notes mirror the lightweight pDynamo3 script templates under
`scripts/pDynamo3/`.

The examples are intentionally template-based. They do not include molecular
files, trajectories, binaries, or unpublished research data. Adapt them with
local inputs and cite pDynamo3 when using it in real work.
