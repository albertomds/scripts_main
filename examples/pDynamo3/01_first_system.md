# First pDynamo3 System

Use `scripts/pDynamo3/01_first_system/run_example.py` as a lightweight starting
point for checking a local tutorial folder before adding real pDynamo3 system
loading code.

Run from the tutorial folder:

```bash
python run_example.py
```

Edit the script before using it for real calculations. Local pDynamo3 imports,
input paths, force-field choices, and environment variables may differ between
installations.
